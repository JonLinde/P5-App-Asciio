# Keyboard

Asciio has many bindings which let you work effectively, a list of all the bindings can be found in the ***Bindings*** section.

| Example |
| ------- |
|         |

| action                | binding            |
| -----                 | --------           |
| insert a box          | ib                 |
| resize it             | 2, 4               |
| move it               | hjkl/Arrows        |
| insert a new box      | ib                 |
| insert an arrow       | ia                 |
| connect other box     | Alt+A, hjkl/Arrows |
| escape arrow end mode | Escape             |
| select both boxes     | V                  |
| align them            | A, t               |
